Calculates the final exponentiation of the optimal ate pairing (hard part).

It calculates a fixed multiple of the actual pairing, which is a valid thing to do. It is a modified version from the final exponentiation presented by Fuentes-Castaneda which takes one more frobenius action and one more multiplication but saves one temporary variable.
